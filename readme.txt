== Installation ==
To install POWr Countdown Timer
<br><br>
1. Upload 'powr-countdown-timer.php' to the '/wp-content/plugins/' directory
<br>
2. Activate the plugin through the 'Plugins' menu in WordPress
<br><br>
To add POWr Countdown Timer as a widget, go to **Appearance->Widgets**, and drag POWr Countdown Timer to where you'd like it to appear.
<br><br>
To add Countdown Timer within a post or a page, simply write the shortcode **[powr-countdown-timer label='MY LABEL']** (including the brackets) where you'd like your Countdown Timer to appear. Then visit your page!
<br><br>
Want a more detailed tutorial? Visit [How to add Countdown Timer to your Wordpress site](http://www.powr.io/tutorials/how-to-add-countdown-timer-plugin-to-your-wordpress-site)


== Frequently Asked Questions ==

= Do I need to create a POWr Account? = Yes, you need a POWr Account.  Your account allows you to edit plugins right in your live webpage, and to recreate plugins across many different sites.

= How do I add Countdown Timer to a page or blog post? = Add the shortcode **[powr-countdown-timer label='Enter any label here']** to any page or post. Then visit your live site to edit.

= Can I add more than one Countdown Timer? = Yes, absolutely!  To clone your Countdown Timer, just add the exact same shortcode to pages or posts.  To create a new version of Countdown Timer, add a different shortcode to each page or post, for example: **[powr-countdown-timer label='first form']**, and **[powr-countdown-timer label='second form']**.

= Will my users see the settings icon in the corner of the Countdown Timer? = No! You only see the settings icon because you own the plugins. To temporarily hide it, click on your page and then type 'p + down arrow' (the letter p followed by the down arrow). You can show the icon again by typing 'p + up-arrow'.

= I can't see the settings icon anymore, what happened? = You are just logged out of POWr.io. You can show the icon again by typing 'p + up-arrow'.

= Is POWr Countdown Timer Free? = Yes, the basic version of Countdown Timer is completely free!
